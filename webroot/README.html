The data directory should mount a data component (e.g. S3, NFS, SMB, etc) containing:

  • the data/rdf directory that contains linked data expressed in RDF turle according with the Mopso Language Profile;
  • an optional data/raw directory that contains raw data extractions that require a transformation in order to be loaded by the knowledge graph;

For more info about Legacy proxy and data components, refer to the Mopso architecture documentation
